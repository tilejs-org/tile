## 1. Defina a interface e o schema

```ts title="src/database/schema/member.ts"
import { Schema } from "@tile.js/database";

export interface Member {
  _id?: string;
  name: string;
  email: string;
  verified?: boolean;
}

export const memberSchema = new Schema<Member>({
  name: {
    type: String,
    required: true,
    trim: true,
  },
  email: {
    type: String,
    required: true,
    unique: true,
    lowercase: true,
  },
  verified: {
    type: Boolean,
    default: false,
  },
});
```

## 2. Configure a collection e centralize a database

```ts title="src/database/index.ts"
import { Database } from "@tile.js/database";
import { memberSchema, type Member } from "./schema/member.ts";

const connection = new Database({
  dbName: "app",
});

export const database = Object.assign(connection, {
  member: connection.collection<Member>("member", memberSchema),
});
```

## 3. Use a database na aplicação

```ts title="src/index.ts"
import { database } from "./database/index.ts";

await database.connect();

const member = await database.member.create({
  name: "Ada Lovelace",
  email: "ADA@EXAMPLE.COM",
});

const found = await database.member.findOne({
  email: "ada@example.com",
});

await database.member.updateOne(
  { _id: member._id },
  {
    $set: { verified: true },
  },
);

console.log(found);

await database.disconnect();
```

## Resultado esperado

- `name` será normalizado com `trim`
- `email` será salvo em minúsculas por causa de `lowercase`
- `verified` receberá `false` por `default`
- `_id`, `__v` e timestamps serão adicionados automaticamente por padrão

<Callout type="tip" title="No modo local">
Com o adapter padrão, os dados ficam em `.tile/database/<dbName>/<collection>/`. Você não precisa criar essa pasta manualmente.
</Callout>
