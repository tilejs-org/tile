# CRUD

Após criar uma collection, você pode inserir, consultar, atualizar e remover documentos.

Todos os métodos são assíncronos e retornam uma `Promise`.

<Callout type="info">
Os filtros são simples e baseados em igualdade rasa entre campos e valores.
</Callout>

---

## Collection de exemplo

```ts title="database.ts"
import { Database, Schema } from "@tile.js/database";

interface Member {
  _id?: string;
  name: string;
  email: string;
  age?: number;
  verified?: boolean;
  note?: string;
}

const memberSchema = new Schema<Member>({
  name: {
    type: String,
    required: true,
  },
  email: {
    type: String,
    unique: true,
    lowercase: true,
  },
  age: Number,
  verified: {
    type: Boolean,
    default: false,
  },
  note: String,
});

const connection = new Database();

export const database = Object.assign(connection, {
  members: connection.collection<Member>("members", memberSchema),
});
```

---

# Inserindo documentos

## create()

Insere um único documento.

```ts
const member = await database.members.create({
  name: "Israel",
  email: "ISRAEL@GMAIL.COM",
  age: 21,
});

console.log(member);
```

Retorno:

```ts
{
  _id: "68526dba05db8ec36dcb33d8",
  name: "Israel",
  email: "israel@gmail.com",
  age: 21,
  verified: false,
  __v: 0,
  createdAt: Date,
  updatedAt: Date
}
```

---

# Buscando documentos

## findOne()

Retorna o primeiro documento encontrado.

```ts
const member = await database.members.findOne({
  email: "israel@gmail.com",
});
```

---

## find()

Retorna todos os documentos que correspondem ao filtro.

```ts
const members = await database.members.find({
  verified: true,
});
```

Também é possível buscar todos os documentos:

```ts
const members = await database.members.find();
```

---

## findOneByID() / findOneById()

Buscam um documento pelo `_id`.

```ts
const member = await database.members.findOneByID("68526dba05db8ec36dcb33d8");
```

Alias:

```ts
const member = await database.members.findOneById("68526dba05db8ec36dcb33d8");
```

---

## countDocuments()

Conta quantos documentos existem na collection.

```ts
const total = await database.members.countDocuments();
```

Com filtro:

```ts
const verifiedTotal = await database.members.countDocuments({
  verified: true,
});
```

---

# Atualizando documentos

## updateOne()

Atualiza o primeiro documento correspondente e retorna o documento atualizado.

```ts
const updated = await database.members.updateOne(
  {
    email: "israel@gmail.com",
  },
  {
    verified: true,
  },
);
```

Também é possível utilizar operadores.

```ts
const updated = await database.members.updateOne(
  {
    email: "israel@gmail.com",
  },
  {
    $set: {
      verified: true,
      name: "Israel Rabbi",
    },
    $inc: {
      age: 1,
    },
    $unset: ["note"],
  },
);
```

---

## upsert()

Atualiza o primeiro documento encontrado ou cria um novo quando nada é encontrado.

```ts
const member = await database.members.upsert(
  {
    email: "israel@gmail.com",
  },
  {
    name: "Israel",
    verified: true,
  },
);
```

# Removendo documentos

## deleteOne()

Remove o primeiro documento encontrado.

```ts
await database.members.deleteOne({
  email: "israel@gmail.com",
});
```

---

## deleteMany()

Remove vários documentos.

```ts
await database.members.deleteMany({
  verified: false,
});
```

Remover todos:

```ts
await database.members.deleteMany();
```

---

# Paginação

## paginate()

Retorna documentos paginados.

```ts
const result = await database.members.paginate({
  page: 1,
  limit: 10,
});
```

Também é possível ordenar:

```ts
const result = await database.members.paginate({
  page: 1,
  limit: 10,
  sort: {
    createdAt: "desc",
  },
});
```

Retorno:

```ts
{
  items: [...],
  total: 53,
  page: 1,
  limit: 10,
  pages: 6
}
```
