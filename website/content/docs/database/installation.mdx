# Instalação

O **@tile.js/database** instala apenas o núcleo da lib e o adapter padrão local em BSON.

Adapters externos como **MongoDB** e **Mongoose** usam dependências fornecidas pelo consumidor final.

---

## Instale o pacote principal

<Tabs items={["bun", "npm", "pnpm", "yarn"]}>
  <Tab value="bun">

```bash
bun add @tile.js/database
```

  </Tab>

  <Tab value="npm">

```bash
npm install @tile.js/database
```

  </Tab>

  <Tab value="pnpm">

```bash
pnpm add @tile.js/database
```

  </Tab>

  <Tab value="yarn">

```bash
yarn add @tile.js/database
```

  </Tab>
</Tabs>

---

## Instale adapters opcionais

Se você quiser usar adapters remotos, instale também a dependência correspondente no seu projeto.

<Tabs items={["mongodb", "mongoose"]}>
  <Tab value="mongodb">

```bash
npm install mongodb
```

  </Tab>

  <Tab value="mongoose">

```bash
npm install mongoose
```

  </Tab>
</Tabs>

<Callout type="info">
O pacote **@tile.js/database** não instala `mongodb` nem `mongoose` automaticamente. Essas dependências são responsabilidade do projeto consumidor.
</Callout>

---

## Requisitos

| Requisito | Versão |
| ---------- | ------ |
| Node.js | >= 24.x |
| Bun | >= 1.x |
| TypeScript | >= 5.x |

---

## Verificando a instalação

Crie um arquivo `index.ts` e importe o pacote:

```ts title="index.ts"
import { Database, Schema } from "@tile.js/database";

console.log("Tile.JS Database instalado com sucesso!");
```

Execute:

<Tabs items={["bun", "tsx"]}>
  <Tab value="bun">

```bash
bun index.ts
```

  </Tab>

  <Tab value="tsx">

```bash
npx tsx index.ts
```

  </Tab>
</Tabs>

Se nenhuma mensagem de erro aparecer, a instalação foi concluída corretamente.

---

## O que é criado no modo padrão

O adapter local padrão cria a estrutura `.tile/database/...` automaticamente na primeira vez que uma collection é usada.

<Files>
  <Folder name="src" defaultOpen>
    <File name="index.ts" />
  </Folder>

  <File name="package.json" />
  <File name="tsconfig.json" />
</Files>

Após a primeira operação local, a pasta `.tile` será criada automaticamente.

<Callout type="tip">
No adapter padrão, os documentos são armazenados em **BSON** e os metadados/índices em **JSON**.
</Callout>
