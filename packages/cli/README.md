# Tile.JS CLI

## Instalações


Node.js

```bash
npm install @tile.js/cli
```

Linux

```bash
curl -fsSL https://tile.js.org/install.sh | sh
```

Windows

```bash
irm https://tile.js.org/install.ps1 | iex
```
